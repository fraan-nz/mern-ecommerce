---

#### Clonar Repositorio

```
$ git clone https://github.com/fraan-nz/mern-ecommerce.git
$ cd mern-ecommerce
```

#### Run Backend

```
$ cd backend
$ npm install
$ npm start
```

#### Run Frontend

```
# abrir una nueva terminal
$ cd frontend
$ npm install
$ npm start
```

#### Configurar env

```
# crear .env en la carpeta backend
MONGODB_URI=
JWT_SECRET=
PAYPAL_CLIENT_ID=
```
